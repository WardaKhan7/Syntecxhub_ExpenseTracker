# Expense Tracker

A full-stack Expense Tracker application that helps users manage and track their daily expenses easily.

---

## Features

- Add new expenses
- View a list of expenses
- Delete expenses
- Track total spending
- Responsive user interface
- Data fetched from API
- Optimized performance using React Hooks

---

## Technologies Used

### Frontend
- React.js
- CSS
- Tailwind CSS / Bootstrap

### Backend
- Node.js
- Express.js

### Database
- MongoDB

---

## React Hooks Used

- **useState** – manage form inputs and expenses
- **useEffect** – fetch expense data from API
- **useRef** – manage form focus
- **useMemo** – optimize expensive calculations
- **useCallback** – optimize functions

## Installation

1. Clone the repository
git clone https://github.com/WardaKhan7/Syntecxhub_ExpenseTracker.git

3. Navigate to the project folder
cd Syntecxhub_ExpenseTracker

3. Install dependencies
npm install

4. Run the application
npm start
